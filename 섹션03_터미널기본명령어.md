# 섹션 3. 터미널 기본 명령어

> **이 섹션을 마치면**: Claude Code를 사용하는 데 필요한 모든 터미널 명령어를 다룰 수 있습니다.

---

## 3-1. 필수 명령어 20선

### 탐색 명령어

```bash
# 현재 위치 확인 (Print Working Directory)
pwd

# 폴더 내용 보기 (List)
ls
ls -l      # 상세 정보 포함
ls -la     # 숨김 파일까지 포함

# 폴더 이동 (Change Directory)
cd 폴더명
cd ..          # 상위 폴더
cd ~           # 홈 폴더
cd ~/Documents # 직접 경로 지정
```

### 파일/폴더 생성 및 삭제

```bash
# 폴더 만들기 (MaKe DIRectory)
mkdir 프로젝트이름
mkdir -p a/b/c    # 중간 폴더까지 한 번에 생성

# 파일 만들기
touch 파일명.txt

# 파일/폴더 삭제 (주의!)
rm 파일명.txt           # 파일 삭제
rm -r 폴더명/           # 폴더 전체 삭제 (복구 불가)
```

> ⚠️ `rm -rf` 명령어는 폴더를 완전히 삭제합니다. 휴지통에 가지 않습니다.

### 파일 내용 보기

```bash
# 파일 내용 출력
cat 파일명.txt

# 파일 앞부분만 보기
head -20 파일명.txt    # 앞 20줄

# 파일 끝부분만 보기
tail -20 파일명.txt    # 마지막 20줄
```

### 복사 및 이동

```bash
# 파일 복사 (Copy)
cp 원본.txt 복사본.txt
cp -r 원본폴더/ 복사폴더/    # 폴더 복사

# 파일 이동/이름 변경 (Move)
mv 파일.txt 새이름.txt        # 이름 변경
mv 파일.txt ~/Documents/      # 다른 위치로 이동
```

## 3-2. 자주 쓰는 개발 명령어

### 패키지 관리 (Node.js)

```bash
# Node.js 버전 확인
node --version

# npm 버전 확인
npm --version

# 패키지 설치
npm install 패키지명
npm install -g 패키지명    # 전역 설치

# 패키지 실행
npm run dev     # 개발 서버 시작
npm run build   # 빌드
npm test        # 테스트 실행
```

### Git 기본 (자세한 내용은 섹션 8)

```bash
# Git 버전 확인
git --version

# 저장소 상태 확인
git status

# 변경사항 저장
git add .
git commit -m "변경 내용 설명"
```

## 3-3. 실무에서 유용한 명령어

### 검색

```bash
# 특정 텍스트를 파일에서 찾기
grep "검색어" 파일명.txt
grep -r "검색어" .   # 현재 폴더 전체에서 재귀 검색

# 파일 찾기
find . -name "*.md"    # 현재 폴더에서 .md 파일 찾기
```

### 프로세스 관리

```bash
# 실행 중인 프로세스 목록
ps aux

# 특정 포트 사용 확인 (서버 충돌 시 유용)
lsof -i :3000

# 프로세스 강제 종료 (포트 번호로)
kill -9 $(lsof -t -i :3000)
```

### 환경 변수

```bash
# 환경 변수 확인
echo $PATH

# 임시 환경 변수 설정
export MY_VAR="값"
```

## 3-4. 자주 실수하는 것들

### 실수 1: 경로 오타
```bash
# 잘못된 예
cd Documnets    # 오타

# 해결: Tab으로 자동 완성
cd Doc[Tab]     # → cd Documents/
```

### 실수 2: 명령어 실행 위치 착각
```bash
# 항상 현재 위치 확인하고 시작
pwd

# 프로젝트 폴더 안에서 실행해야 하는 명령들
cd ~/Documents/my-project
npm install
```

### 실수 3: 권한 오류
```bash
# "Permission denied" 에러 시
sudo 명령어    # 관리자 권한으로 실행
# 비밀번호 입력 필요 (입력 시 화면에 안 보임, 정상)
```

## 3-5. 핵심 명령어 치트시트

| 명령어 | 기능 | 예시 |
|--------|------|------|
| `pwd` | 현재 위치 | `pwd` |
| `ls` | 목록 보기 | `ls -la` |
| `cd` | 폴더 이동 | `cd ~/Documents` |
| `mkdir` | 폴더 생성 | `mkdir my-app` |
| `touch` | 파일 생성 | `touch index.html` |
| `cp` | 복사 | `cp a.txt b.txt` |
| `mv` | 이동/이름변경 | `mv old.txt new.txt` |
| `rm` | 삭제 | `rm file.txt` |
| `cat` | 파일 내용 보기 | `cat README.md` |
| `grep` | 텍스트 검색 | `grep "hello" *.js` |
| `npm install` | 패키지 설치 | `npm install` |
| `git status` | Git 상태 | `git status` |

## 3-6. 실습 미션

### 미션: 프로젝트 폴더 만들기

```bash
# 1. Documents 폴더로 이동
cd ~/Documents

# 2. 학습용 폴더 만들기
mkdir claude-practice

# 3. 폴더로 이동
cd claude-practice

# 4. 위치 확인
pwd

# 5. 연습용 파일 만들기
touch hello.txt
touch notes.md

# 6. 목록 확인
ls -la
```

성공하면 이런 화면이 나와야 합니다:
```
-rw-r--r--  1 user  staff  0 Mar 31 09:00 hello.txt
-rw-r--r--  1 user  staff  0 Mar 31 09:00 notes.md
```
