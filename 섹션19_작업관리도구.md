# 섹션 19. 작업 관리 도구

> **이 섹션을 마치면**: Claude Code의 내장 Todo 시스템으로 복잡한 작업을 체계적으로 관리할 수 있습니다.

---

## Claude Code의 작업 관리

### TodoWrite 도구

Claude Code는 내부적으로 할 일 목록을 만들고 추적합니다. 복잡한 작업을 요청하면 자동으로 단계를 나눠서 진행합니다.

```
> SmartQuote의 견적서 PDF 기능을 구현해줘.
  단계적으로 진행하고 각 단계 완료 시 알려줘.

Claude가 자동으로:
[✓] 1. PDF 라이브러리 설치 (react-pdf)
[✓] 2. 견적서 PDF 템플릿 컴포넌트 작성
[✓] 3. PDF 다운로드 API 엔드포인트 생성
[ ] 4. 프론트엔드 다운로드 버튼 연결
[ ] 5. 테스트
```

## Ctrl + T: 작업 목록 보기

```
Ctrl + T    ← 현재 작업 진행 상황 토글
```

긴 작업 중에 언제든 현재 상태를 확인할 수 있습니다.

---

## 백그라운드 작업 (Ctrl + B)

시간이 오래 걸리는 작업을 백그라운드로 전환:

```
> npm 의존성 전체를 업그레이드하고 호환성 테스트 실행해줘

(오래 걸릴 것 같으면)
Ctrl + B    ← 백그라운드로 전환

→ 다른 작업 계속하다가
→ 완료 알림 받으면 결과 확인
```

## ROADMAP.md와 연계

큰 프로젝트는 ROADMAP.md로 전체 계획을 관리합니다.

```
> ROADMAP.md를 만들어줘.
  SmartQuote 프로젝트의 전체 개발 계획을:
  - Phase 1: MVP (2주)
  - Phase 2: 결제 연동 (1주)
  - Phase 3: 팀 기능 (2주)
  로 구성하고, 각 Phase의 세부 Task를 체크박스로 정리해줘.
```
