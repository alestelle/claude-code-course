# Claude Code 완전 정복 커리큘럼

> **대상**: IT 기획 경력자로서 AI 코딩 도구를 실무에 활용하고 싶은 분
> **목표**: Claude Code를 처음부터 실전 수준까지, 기획자의 시선으로 배웁니다

---

## 학습자 안내

이 커리큘럼은 다음과 같은 분을 위해 설계되었습니다:
- 코딩 경험이 없어도 괜찮습니다
- IT 서비스 기획 경험이 있다면 더 빠르게 이해할 수 있습니다
- "왜 이게 필요한가"를 먼저 이해하고, "어떻게 쓰는가"를 배웁니다
