# 섹션 2. 터미널 기초 지식과 활용법

> **이 섹션을 마치면**: 터미널을 두려움 없이 열고, 기본 조작을 할 수 있습니다.

---

## 2-1. 터미널이란 무엇인가

### 쉬운 비유로 이해하기

```
[마우스로 클릭하는 일반 방식]
바탕화면 → 폴더 아이콘 더블클릭 → 파일 찾기 → 오른쪽 클릭 → 열기

[터미널 방식]
> open ~/Documents/my-file.txt  (끝)
```

터미널은 **컴퓨터에게 직접 텍스트로 명령을 내리는 창구**입니다.

마우스 UI가 생기기 전, 모든 컴퓨터 조작은 이렇게 이루어졌습니다. 지금도 개발자들이 터미널을 쓰는 이유는:
- **더 빠르고** (반복 작업을 한 줄로)
- **더 강력하고** (GUI로 못 하는 것도 가능)
- **자동화가 됩니다** (스크립트로 반복 실행)

> Claude Code는 **터미널에서 실행**됩니다. 터미널을 열 줄 알아야 합니다.

---

## 2-2. 터미널 열기 (macOS 기준)

### 방법 1: Spotlight 검색
```
Cmd + Space → "터미널" 입력 → Enter
```

### 방법 2: Launchpad
```
Launchpad → 기타 → 터미널
```

### 방법 3: Finder
```
응용 프로그램 → 유틸리티 → 터미널
```

---

## 2-3. 터미널 화면 읽기

터미널을 열면 이런 화면이 나타납니다:

```
user@MacBook-Pro ~ %  _
│      │           │  └─ 커서 (입력 대기 중)
│      │           └─ 현재 위치 (~ = 홈 폴더)
│      └─ 컴퓨터 이름
└─ 사용자 이름
```

### 중요한 기호들

| 기호 | 의미 |
|------|------|
| `~` | 홈 폴더 (예: /Users/홍길동) |
| `/` | 최상위 폴더 (루트) |
| `%` 또는 `$` | 명령 입력 대기 중 |
| `>` | Claude Code 입력 대기 중 |

---

## 2-4. 터미널 기본 조작

### 명령 실행
```bash
# 텍스트 입력 후 Enter 키 누르기
ls          # 현재 폴더 내용 보기
```

### 작업 취소
```
Ctrl + C    # 실행 중인 작업 강제 중단
```

### 화면 지우기
```
Ctrl + L    # 화면 깨끗하게 (기록은 유지)
clear       # 같은 효과
```

### 이전 명령 불러오기
```
↑ 위 화살표    # 이전에 입력한 명령 불러오기
↓ 아래 화살표  # 다음 명령으로 이동
```

### 자동 완성
```
Tab 키    # 폴더/파일 이름 자동 완성
          # 예: cd Doc[Tab] → cd Documents/
```

---

## 2-5. 폴더 구조 이해하기

컴퓨터의 파일 구조는 **나무처럼 계층적**입니다.

```
/ (최상위)
├── Users/
│   └── 홍길동/          ← ~ (홈 폴더)
│       ├── Documents/
│       ├── Downloads/
│       └── Desktop/
├── Applications/
└── System/
```

### 경로 표현 방식

| 경로 | 설명 |
|------|------|
| `/Users/홍길동/Documents` | 절대 경로 (최상위부터) |
| `~/Documents` | 홈 폴더 기준 단축 표현 |
| `./my-project` | 현재 폴더 기준 (상대 경로) |
| `../` | 상위 폴더 |

---

## 2-6. 실습: 터미널로 폴더 탐색하기

```bash
# 1. 현재 위치 확인
pwd
# 출력 예: /Users/홍길동

# 2. 현재 폴더 내용 보기
ls

# 3. Documents 폴더로 이동
cd Documents

# 4. 다시 위치 확인
pwd
# 출력 예: /Users/홍길동/Documents

# 5. 상위 폴더로 돌아가기
cd ..

# 6. 홈 폴더로 한 번에 가기
cd ~
```

## 핵심 정리

```
터미널 = 텍스트로 컴퓨터에 명령하는 창구
Claude Code = 터미널에서 실행되는 AI 도구
~ = 내 홈 폴더 (출발점)
Tab = 자동 완성 (타이핑 절약)
Ctrl+C = 긴급 취소 (막히면 이거)
```
