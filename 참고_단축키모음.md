# 단축키 빠른 참조

## 가장 중요한 단축키 (매일 사용)

| 단축키 | 기능 |
|--------|------|
| `Ctrl + C` | 현재 작업 취소/중단 |
| `Ctrl + D` | Claude Code 종료 |
| `Esc + Esc` | 이전 상태로 되돌리기 |
| `Shift + Tab` | 권한 모드 순환 전환 |
| `↑` / `↓` | 이전/다음 명령 히스토리 |
| `Tab` | 자동 완성 |

## 모델 및 모드

| 단축키 | 기능 |
|--------|------|
| `Option + P` | 모델 전환 |
| `Option + T` | 확장 사고 토글 |
| `Option + O` | 빠른 모드 토글 |

## 화면 및 작업

| 단축키 | 기능 |
|--------|------|
| `Ctrl + L` | 화면 지우기 |
| `Ctrl + B` | 백그라운드로 전환 |
| `Ctrl + T` | 작업 목록 토글 |
| `Ctrl + O` | 상세 출력 모드 |

## 텍스트 입력

| 단축키 | 기능 |
|--------|------|
| `Option + Enter` | 줄바꿈 (멀티라인) |
| `Ctrl + R` | 명령 히스토리 검색 |

## 특수 입력 접두사

| 접두사 | 기능 |
|--------|------|
| `/` | 슬래시 명령어 |
| `!` | 셸 명령어 직접 실행 |
| `@` | 파일 참조 |
