# 섹션 17. [프로젝트1] MCP Server 설치

> **이 섹션을 마치면**: 프로젝트에 필요한 MCP 서버를 설치하고 Claude Code와 연동할 수 있습니다.

---

## Starter Kit에 MCP 연동하기

섹션 10에서 MCP 개념을 배웠습니다. 여기서는 실제 프로젝트에 연결하는 방법을 실습합니다.

---

## 스포츠 검색 서비스에 유용한 MCP 서버

| MCP 서버 | 용도 |
|---------|------|
| `brave-search` | 스포츠 최신 뉴스 검색 |
| `github` | 이슈 관리, 코드 배포 |
| `postgresql` | DB 직접 조회 |
| `filesystem` | 파일 시스템 확장 접근 |

---

## MCP 서버 설치 실습

### 1. brave-search 설치

```bash
# Brave Search API 키 발급
# brave.com/search/api 에서 무료 계정 생성

# .claude/mcp.json 생성
claude
> .claude/mcp.json 파일을 만들어줘.
  brave-search MCP 서버를 추가하는데
  API 키는 환경변수 BRAVE_API_KEY로 읽어줘.
```

### 2. PostgreSQL MCP 설치

```json
{
  "mcpServers": {
    "postgresql": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres",
               "postgresql://localhost:5432/sports_db"]
    }
  }
}
```

활용:
```
> (postgresql 연결 후)
  players 테이블에서 포지션이 '공격수'인 선수 목록 조회해줘
  결과를 JSON으로 보여줘
```

---

## MCP 서버 + 스킬 조합

MCP 서버를 스킬과 함께 쓰면 더 강력합니다:

```markdown
# .claude/skills/sports-news.md
---
description: 최신 스포츠 뉴스를 검색하고 DB에 저장합니다
---

brave-search MCP를 사용해서:
1. "${키워드} 최신 스포츠 뉴스"를 검색
2. 상위 5개 결과를 요약
3. news 테이블에 저장 (중복 제외)
```

실행:
```
> /sports-news 손흥민
> /sports-news K리그 이번주 결과
```

---

## MCP 연결 확인

```
> /mcp

출력 예:
✅ brave-search  - 웹 검색 도구 활성화
✅ postgresql    - DB 조회 도구 활성화
```

---

## 실습 미션

```
> 스포츠 검색 서비스에 다음 MCP를 연결해줘:
  1. brave-search (API 키는 .env에서 읽기)
  2. postgresql (로컬 DB 연결)

  연결 후 테스트:
  - "K리그 최신 순위" 검색
  - 로컬 DB에서 선수 목록 조회

  스킬도 만들어줘: /news [키워드] → 최신 스포츠 뉴스 검색
```
