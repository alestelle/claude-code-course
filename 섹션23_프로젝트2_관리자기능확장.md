# 섹션 23. [프로젝트2] 관리자 기능 확장

> **이 섹션을 마치면**: 관리자가 견적서를 관리할 수 있는 대시보드를 완성합니다.

---

## 관리자 기능 범위

```
관리자 대시보드 기능:
✅ 견적서 목록 조회 (필터/검색)
✅ 견적서 상태 변경 (대기→확인→완료)
✅ 통계 현황 (이번 달 견적 수, 금액 합계)
✅ 고객 목록
✅ 비밀번호 보호
```

---

## 관리자 페이지 개발

### 인증 설정

```
> /admin 경로를 비밀번호로 보호해줘.
  Next.js middleware를 사용하고,
  환경변수 ADMIN_PASSWORD로 설정 가능하게 해줘.
  간단한 세션 쿠키 방식으로 구현해줘.
```

### 견적서 목록 페이지

```
> /admin/quotes 페이지를 만들어줘.
  Notion DB에서 견적서 목록을 가져와서 표시하고:
  - 고객명, 날짜, 금액, 상태 컬럼
  - 상태별 필터 (전체/대기/확인/완료)
  - 클릭 시 상세 보기
  - 상태 변경 버튼
```

### 통계 대시보드

```
> /admin 메인 페이지에 통계 위젯을 추가해줘:
  - 이번 달 견적서 수
  - 이번 달 총 견적 금액
  - 상태별 건수 (파이 차트)
  - 최근 7일 견적서 추이 (라인 차트)
```

```
> /admin 페이지에서 관리자가 Claude Code에게
  Notion 데이터를 조회할 수 있는 MCP 연동을 추가해줘.

예: 관리자가 물어볼 수 있는 것:

> "3일째 대기 중인 견적서 있어?"
> "가장 많이 쓰는 품목 Top 5는?"
```
