# 문제 해결 가이드

## 자주 겪는 문제

### Claude Code가 실행이 안 됨

```bash
# 설치 확인
claude --version

# 재설치
npm install -g @anthropic-ai/claude-code

# 로그인 재시도
claude login
```

### "Permission denied" 에러

```bash
# 권한 확인
ls -la ~/.claude/

# npm 전역 설치 권한 문제
sudo npm install -g @anthropic-ai/claude-code
```

### CLAUDE.md가 읽히지 않음

```
> /memory
# CLAUDE.md 위치와 내용 확인
```

### 컨텍스트 부족 (대화가 길어질 때)

```
> /compact
# 대화 압축

> /context
# 현재 사용량 확인
```

### 훅이 실행되지 않음

```
> /hooks
# 훅 목록 확인

# 훅 스크립트 실행 권한 확인
chmod +x .claude/hooks/*.sh
```

### MCP 연결 실패

```
> /mcp
# 서버 상태 확인

# API 키 환경변수 확인
echo $BRAVE_API_KEY
```

### npm 명령어가 없다고 할 때

```bash
# Node.js 설치 확인
node --version
npm --version

# 설치 안 됐으면
# nodejs.org 에서 LTS 버전 다운로드
```

### git 명령어 오류

```bash
# Git 설치 확인
git --version

# 사용자 정보 설정 안 됐을 때
git config --global user.name "이름"
git config --global user.email "이메일"
```
