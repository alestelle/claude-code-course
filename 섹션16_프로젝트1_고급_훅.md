# 섹션 16. [프로젝트1] 고급 - 훅(Hooks)

> **이 섹션을 마치면**: 특정 이벤트가 발생할 때 자동으로 실행되는 자동화 규칙을 만들 수 있습니다.

---

## 훅(Hooks)이란?

```
훅 = "이런 일이 생기면, 이걸 자동으로 실행해줘"

스킬과의 차이:
스킬 = 내가 직접 /명령어 로 실행
훅   = 특정 이벤트 발생 시 자동 실행

비유:
스킬 = 버튼 (누를 때 실행)
훅   = 센서 (조건 충족 시 자동 실행)
```

---

## 훅 이벤트 종류

| 이벤트 | 언제 발생 | 활용 예시 |
|--------|----------|----------|
| `PostToolUse` | 도구 실행 후 | 파일 저장 후 자동 포맷팅 |
| `PreToolUse` | 도구 실행 전 | 위험한 명령어 차단 |
| `SessionStart` | 세션 시작 시 | 환경 설정 자동 로드 |
| `Stop` | Claude 응답 완료 후 | 작업 결과 검증 |
| `Notification` | 알림 발생 시 | 데스크탑 알림 |
| `UserPromptSubmit` | 요청 제출 시 | 입력 검증 |

---

## 실전 훅 예시

### 예시 1: 코드 저장 시 자동 포맷팅

```json
// .claude/settings.json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [
          {
            "type": "command",
            "command": "npx prettier --write $CLAUDE_FILE_PATH"
          }
        ]
      }
    ]
  }
}
```

파일을 수정할 때마다 자동으로 코드 포맷팅!

### 예시 2: .env 파일 보호

```json
{
  "hooks": {
    "PreToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [
          {
            "type": "command",
            "command": "if [[ \"$CLAUDE_FILE_PATH\" == *\".env\"* ]]; then echo '.env 파일은 수정할 수 없습니다' >&2; exit 2; fi"
          }
        ]
      }
    ]
  }
}
```

### 예시 3: 작업 완료 시 데스크탑 알림

```json
{
  "hooks": {
    "Stop": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "osascript -e 'display notification \"Claude Code 작업 완료!\" with title \"Claude Code\"'"
          }
        ]
      }
    ]
  }
}
```

긴 작업이 끝나면 알림을 받을 수 있습니다.

### 예시 4: 세션 시작 시 브리핑

```json
{
  "hooks": {
    "SessionStart": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "echo \"오늘 날짜: $(date). Git 상태: $(git status --short | wc -l)개 파일 변경됨\""
          }
        ]
      }
    ]
  }
}
```

---

## Claude Code에서 훅 설정하기

```
> .claude/settings.json에 훅을 추가해줘:

  1. TypeScript나 JavaScript 파일을 수정할 때마다
     자동으로 ESLint 검사 실행

  2. 세션이 시작될 때 git status 보여주기

  3. Claude가 작업을 완료할 때마다
     맥 데스크탑 알림 보내기
```


## 실습 미션

```
> 스포츠 검색 서비스 프로젝트에 맞는 훅을 설정해줘:

  1. JS/TS 파일 수정 시 → ESLint 자동 실행
  2. 빌드 실패 시 → 에러 내용 파일로 저장
  3. 작업 완료 시 → 맥 알림
  4. .env 파일 수정 시도 → 차단

  .claude/settings.json에 모두 추가해줘.
```
