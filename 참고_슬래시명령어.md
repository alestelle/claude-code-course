# 슬래시 명령어 전체 목록

## 프로젝트 관리
| 명령어 | 기능 |
|--------|------|
| `/init` | 프로젝트 초기화, CLAUDE.md 생성 |
| `/memory` | 메모리 확인/편집 |
| `/clear` | 대화 기록 초기화 |

## 세션 관리
| 명령어 | 기능 |
|--------|------|
| `/continue` | 이전 세션 이어하기 |
| `/fork-session` | 현재 대화 복사해서 다른 방향 시도 |
| `/compact` | 대화 압축 (컨텍스트 절약) |
| `/context` | 컨텍스트 사용량 확인 |

## 모델 및 설정
| 명령어 | 기능 |
|--------|------|
| `/model` | AI 모델 변경 |
| `/config` | 설정 메뉴 |
| `/theme` | 터미널 테마 변경 |
| `/vim` | vim 모드 토글 |

## 도구
| 명령어 | 기능 |
|--------|------|
| `/mcp` | MCP 서버 관리 |
| `/hooks` | 훅 설정 확인 |
| `/skills` | 스킬 목록 |
| `/agents` | 에이전트 설정 |

## 특수 기능
| 명령어 | 기능 |
|--------|------|
| `/btw` | 컨텍스트 유지하며 빠른 질문 |
| `/simplify` | 코드 품질 검토 |
| `/batch` | 병렬 작업 계획 |
| `/loop [간격] [명령]` | 반복 실행 |
| `/schedule` | 예약 실행 |
| `/feedback` | Anthropic에 피드백 |
