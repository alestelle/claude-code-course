# 섹션 4. 클로드 코드 맛보기 및 초기화

> **이 섹션을 마치면**: Git, Node.js, Claude Code를 설치하고 첫 번째 AI 코딩을 경험합니다.

---

## 4-1. 사전 준비: Git 설치

### Git이 왜 필요한가?

```
Git = 파일의 '변경 이력 관리 시스템'

기획 문서로 비유하면:
- 기획서_v1.docx
- 기획서_v2_수정.docx
- 기획서_v3_최종.docx
- 기획서_v4_진짜최종.docx

→ Git은 이걸 하나의 파일로 관리하면서
   언제든 이전 버전으로 돌아갈 수 있게 해줍니다
```

Claude Code는 파일을 편집할 때 Git으로 변경사항을 추적합니다.

### macOS에 Git 설치

```bash
# 방법 1: Xcode Command Line Tools (권장)
xcode-select --install
# 팝업창에서 "설치" 클릭

# 방법 2: Homebrew 이용
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install git

# 설치 확인
git --version
# 출력 예: git version 2.39.0
```

### Git 초기 설정 (최초 1회)

```bash
# 이름 설정 (커밋 시 표시될 이름)
git config --global user.name "홍길동"

# 이메일 설정
git config --global user.email "hong@example.com"

# 설정 확인
git config --list
```

## 4-2. Node.js 설치

### Node.js가 왜 필요한가?

Claude Code는 Node.js 위에서 실행됩니다. Node.js는 JavaScript를 터미널에서 실행할 수 있게 해주는 런타임 환경입니다.

```bash
# Node.js 공식 사이트에서 LTS 버전 다운로드
# https://nodejs.org (LTS = 안정적인 버전)

# 설치 후 확인
node --version
# 출력 예: v20.11.0

npm --version
# 출력 예: 10.2.4
```

> Node.js 18 이상이면 Claude Code 실행 가능합니다.

## 4-3. Claude Code 설치

```bash
# 전역 설치 (-g 옵션)
npm install -g @anthropic-ai/claude-code

# 설치 확인
claude --version
```

### 로그인

```bash

# Anthropic 계정으로 로그인
# (계정이 없으면 claude.ai에서 가입)
```

## 4-4. 첫 번째 Claude Code 실행

### 프로젝트 폴더 만들고 시작하기

```bash
# 실습 폴더 만들기
mkdir ~/Documents/my-first-claude
cd ~/Documents/my-first-claude

# Claude Code 시작
claude
```

## 4-5. /init 명령어로 프로젝트 초기화

```
> /init
```

`/init`을 실행하면 Claude Code가:
1. 현재 폴더 구조를 분석
2. 프로젝트 성격 파악
3. **CLAUDE.md** 파일 자동 생성

### CLAUDE.md가 왜 중요한가?

```
CLAUDE.md = Claude Code에게 주는 '프로젝트 설명서'

매 대화마다 Claude는 이 파일을 자동으로 읽습니다.
→ 프로젝트 목적, 규칙, 명령어를 여기에 적어두면
   매번 설명하지 않아도 됩니다.
```

## 4-6. 기본 인터페이스 익히기

### 허가 요청 이해하기

Claude Code가 파일을 수정하거나 명령을 실행하기 전에 물어봅니다:

```
Would you like to create index.html?
❯ Yes
  No
  Always allow this type
```

- **Yes**: 이번만 허용
- **No**: 취소
- **Always allow**: 앞으로 이런 작업은 자동 허용

### 작업 취소하기

```
Ctrl + C    # 현재 작업 중단
Esc + Esc   # 이전 상태로 되돌리기 (체크포인트 복원)
```
