## ROADMAP.md의 역할

```
CLAUDE.md   = 프로젝트가 무엇인지 설명
ROADMAP.md  = 어떻게 만들어 나갈지 계획

ROADMAP.md가 있으면:
- Claude Code가 전체 그림을 이해
- 지금 어디까지 왔는지 파악
- 다음에 무엇을 해야 할지 명확
```

## Claude Code로 ROADMAP 관리하기

```

  남은 작업 중 가장 중요한 것 3가지를 추천해줘.
```

### 로드맵 기반 작업 지시

```

  견적서 디자인은 기본 깔끔한 스타일로.
```
